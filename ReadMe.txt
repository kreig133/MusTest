Перед началом использования приложения необходимо скачать и запустить сервер драйвера IE. Скачать можно здесь: http://code.google.com/p/selenium/downloads/list.
Для того, чтобы консоль нормально отображала кириллицу, необходимо выбрать в настройках консоли любой векторный шрифт.
В локальном m2-репозитории должен быть драйвер MS JDBC 4.0 (http://claude.betancourt.us/add-microsoft-sql-jdbc-driver-to-maven/).
